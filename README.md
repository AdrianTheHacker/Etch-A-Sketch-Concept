# Etchasketch Concept
This project is designed to be a concept of what an etchasketch could be. It'll allow for modern features that aid artist ability, and truly allow them to *Draw the Future*.

## Requirements
In no particular order, the requirements are as follows:
- Display has at least 3 colours (white counts as a colour).
- Display can update at least once per second.
- Users can control the drawing cursor of the display using rotary inputs.
- Users can save the current drawing on the display on a laptop.
- Different devices can share a given drawing session to collaborate on designs.
- The product should be at most 77mm by 100mm by 40mm.
